В данной курсовой работе была осуществлена разработка базы данных для предметной области "Птицеферма".

Разработка включала в себя:
- Семантическое описание предметной области
- Разработка логической модели данных
- Определение типов хранимых данных
- Задание ограничений целостности данных
- Построение схемы БД
- Разработка сценария для создания БД и основных объектов структуры БД
- Разработка основных операций реляционной алгебры
- Разработка запросов к БД для реализации информационных потребностей пользователя

Работа выполнялась в среде Microsoft SQL Server.
